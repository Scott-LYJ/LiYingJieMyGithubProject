# merciqiao-vue

> 这是一个vue+elementui的后台管理系统模版框架

## 运行和打包说明

``` bash
# 安装包依赖
npm install

# 运行开发环境 localhost:8080
npm run dev

# 打包生产环境,目标目录dist
npm run build

# 分析打包详情
npm run build --report
```
## 主要技术
 - vue:前端js框架
 - element-ui:跟vue搭配的前端ui框架
 - axios:调用后端接口的ajax工具
 - vuex:全局状态管理工具
 - vue-router:路由工具
 - 等等
